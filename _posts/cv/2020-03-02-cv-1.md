## 1.Anaconda 설치
Python을 활용하여 Opencv를 사용하기 위해 Anaconda를 사용할 것이다. 아나콘다를 사용했을 때 가상환경 설정이 가능하기 때문에 필요한 라이브러리만 상황에 맞게 사용하기가 용이하다.

우선 맥북에서 아나콘다를 설치하기 위해 아나콘다 홈페이지로 들어가준다. 아래 사이트로 들어가면

https://www.anaconda.com/products/individual#download-section

<p align="center"><img src="https://github.com/Changhyun-song/Changhyun-song.github.io/blob/master/_posts/images/cv/opencv1-1.png?raw=1" width = "800" ></p>

여기서 가운데 맥OS에서 command line installer을 사용하여 설치한다. graphic installer을 사용하지 않는 이유는 iterm2를 사용하고 있기 때문에 설치했을 때 기존에 terminal에서는 anaconda를 사용할 수 있지만 기본적으로 사용하고 있는 iterm2 터미널에서는 찾을 수 없다고 나온다. zsh 쉘이 conda를 찾지 못하기 때문에 command line installer을 이용해 다운로드 한다.

<p align="center"><img src="https://github.com/Changhyun-song/Changhyun-song.github.io/blob/master/_posts/images/cv/opencv1-2.png?raw=1" width = "800" ></p>

터미널 창을 열고 Downloads로 들어가 다운로드 받은 Anaconda 파일을 확인한다.

Enter키를 쭉 누르면서 밑으로 내리면 설치 동의 메시지가 나온다. 그 때 Yes 하면 설치가 진행된다. 설치가 완료된 뒤에 초기화할껀지 물어보는데 Yes를 해주면 zshrc 파일도 수정되었다는 것도 확인할 수 있다.

<p align="center"><img src="https://github.com/Changhyun-song/Changhyun-song.github.io/blob/master/_posts/images/cv/opencv1-4.png?raw=1" width = "800" ></p>

## 2.Anaconda 가상환경 설정

터미널 창을 열고 
conda create -n tf python=3.8
conda 명령어를 입력하면 tf 라는 가상환경이 생성되고 python은 3.8 버전으로 생성된다.
가상환경 이름은 원하는 걸로 만들면 되고 python 버전도 원하는 버전에 맞춰서 만들어주면 된다.
<p align="center"><img src="https://github.com/Changhyun-song/Changhyun-song.github.io/blob/master/_posts/images/cv/opencv1-7.png?raw=1" width = "800" ></p>


그 후 터미널 창에
conda activate tf 
해주면 다음과 같이 tf 가상환경으로 바뀌는 것을 확인할 수 있다. 활성화 시켜준 뒤에 원하는 패키지가 있으면 추가로 받으면 된다.

<p align="center"><img src="https://github.com/Changhyun-song/Changhyun-song.github.io/blob/master/_posts/images/cv/opencv1-8.png?raw=1" width = "800" ></p>

여기선 opencv를 받아야 하기 때문에 opencv를 설치해준다.
pip install opencv-python
하면 다음과 같이 opencv 설치가 완료된다.

<p align="center"><img src="https://github.com/Changhyun-song/Changhyun-song.github.io/blob/master/_posts/images/cv/opencv1-9.png?raw=1" width = "800" ></p>

설치가 잘 되었는지 확인하기 위해 pycharm으로 들어가서 tf 가상환경을 설정해준다.
